## 2. 特征工程

### 2.1 Yang-Zhang 波动率计算（标准滚动窗口形式）

Yang-Zhang 波动率是一种结合隔夜波动和日内波动的波动率估计量，能够有效捕捉开盘跳空和日内价格变化。

**公式定义：**
$$\sigma^2_{YZ} = \sigma^2_o + k \cdot \sigma^2_c + (1-k) \cdot \sigma^2_{RS}$$

其中：
- **隔夜收益率方差**：$\sigma^2_o = \text{Var}(r_o)$，$r_o = \ln(O_t / C_{t-1})$
- **日内收益率方差**：$\sigma^2_c = \text{Var}(r_c)$，$r_c = \ln(C_t / O_t)$
- **Rogers-Satchell方差**：$\sigma^2_{RS} = \text{mean}(RS)$，$RS = u(u-c) + d(d-c)$
- **最优权重**：$k = \frac{0.34}{1.34 + \frac{n+1}{n-1}}$
- **滚动窗口**：$n = 20$ 日

**实现代码：**
```python
# 隔夜收益率
r_o = np.log(df['open'] / df['pre_close'])

# 日内收益率
r_c = np.log(df['close'] / df['open'])

# Rogers-Satchell日内波动
u = np.log(df['high'] / df['open'])
d = np.log(df['low'] / df['open'])
c = np.log(df['close'] / df['open'])
rs = u * (u - c) + d * (d - c)

# 滚动窗口参数
n = 20
k = 0.34 / (1.34 + (n + 1) / (n - 1))

# 方差估计
sigma_o2 = r_o.rolling(n).var(ddof=1)
sigma_c2 = r_c.rolling(n).var(ddof=1)
sigma_rs2 = rs.rolling(n).mean()

# Yang-Zhang波动率
yz_var = sigma_o2 + k * sigma_c2 + (1 - k) * sigma_rs2
df['yang_zhang_vol'] = np.sqrt(np.maximum(yz_var, 0))
```

## 4. 基线模型

### 4.1 模型定义

所有基线模型均预测 Yang-Zhang 波动率，确保公平比较：

| 模型 | 描述 | 预测公式 |
|------|------|----------|
| **Naive** | 持续性预测 | $\hat{y}_t = y_{t-1}$ |
| **MA-5** | 5日移动平均 | $\hat{y}_t = \frac{1}{5}\sum_{i=1}^{5} y_{t-i}$ |
| **MA-20** | 20日移动平均 | $\hat{y}_t = \frac{1}{20}\sum_{i=1}^{20} y_{t-i}$ |
| **EWMA-λ0.94** | 指数加权移动平均 | $\hat{y}_t = \lambda \hat{y}_{t-1} + (1-\lambda) y_{t-1}$ |

### 4.2 评估指标

- **MAE**：平均绝对误差
- **RMSE**：均方根误差
- **R²**：决定系数
- **MAPE**：平均绝对百分比误差
- **Direction Accuracy**：方向预测准确率
- **Correlation**：预测值与真实值相关系数
- **Hit Rate**：预测误差在±20%范围内的比例

## 5. 实验结果

### 5.1 模型对比

| 模型 | MAE | RMSE | R² | MAPE(%) | 方向准确率(%) | 相关系数 | 命中率(%) |
|------|-----|------|-----|---------|--------------|----------|----------|
| **LightGBM** | **0.000302** | 0.001287 | 0.895 | **2.45** | **69.51** | 0.952 | **97.77** |
| Naive | 0.000443 | **0.001106** | **0.916** | 4.72 | 57.75 | **0.958** | 96.92 |
| MA-5 | 0.000752 | 0.001599 | 0.825 | 7.78 | 58.52 | 0.911 | 92.96 |
| MA-20 | 0.001654 | 0.002886 | 0.431 | 16.77 | 51.76 | 0.687 | 74.03 |
| EWMA-λ0.94 | 0.001690 | 0.002731 | 0.488 | 17.40 | 54.64 | 0.701 | 70.41 |

### 5.2 结果分析

**LightGBM 表现最优：**
- **MAE 最小**：0.000302，比 Naive 低 32%
- **MAPE 最低**：2.45%，显著优于其他模型
- **方向准确率最高**：69.51%
- **命中率最高**：97.77%

**关于 R² 的说明：**
- Naive 的 R² (0.916) 高于 LightGBM (0.895)，但这并不意味着 Naive 更好
- R² 衡量的是预测值与真实值的方差解释能力，而非预测精度
- 由于目标变量 (20日滚动波动率) 本身非常平滑，任何平滑预测都能获得高 R²
- **MAE 和 MAPE 才是衡量预测精度的可靠指标**

**EWMA 异常分析：**
- EWMA 的 R² (0.73) 和相关系数 (0.70) 较低
- MAE 高达 0.00169，是 LightGBM 的 5.6 倍
- 原因：λ=0.94 使预测过度平滑，无法及时响应波动变化

### 6.3 泛化能力分析

```
泛化能力排序：
上证50 > 沪深300 > 中证500 > 深证成指 > 中证1000 > 创业板指
   ↓
大盘蓝筹指数泛化好，创业板泛化差
```

**原因分析**：

| 指数类型 | 泛化表现 | MAE增量 | 原因 |
|----------|----------|---------|------|
| 超大盘（上证50） | 优秀 | 16% | 与上证指数成分重叠高，波动特性相似 |
| 大盘蓝筹（沪深300） | 良好 | 35% | 蓝筹股主导，波动相对稳定 |
| 中盘（中证500） | 中等 | 83% | 成分股差异增大，波动特性有差异 |
| 中小板（中证1000） | 较弱 | 151% | 小盘股波动更剧烈，与大盘特性差异大 |
| 创业板 | 最弱 | 312% | 高成长股占比高，波动特性完全不同 |

### 6.4 泛化能力边界

**模型适用范围**：
- ✅ **推荐使用**：上证指数、上证50、沪深300
- ⚠️ **谨慎使用**：中证500、深证成指、中证1000
- ❌ **不推荐使用**：创业板指

**R² vs MAE 的启示**：
- R² 下降幅度小（0.92 → 0.79，仅降14%）
- MAE 增加幅度大（平均增加117%）
- 说明预测趋势方向正确，但精度明显下降

### 6.5 改进建议

1. **为创业板单独训练模型**：波动特性差异太大
2. **多指数联合训练**：增加数据多样性，提升泛化能力
3. **添加市场风格特征**：区分大盘/小盘、成长/价值风格

## 7. 结论

### 7.1 主要发现

1. **Yang-Zhang 波动率是有效的波动率估计量**，结合隔夜和日内波动，能全面捕捉市场波动特征

2. **LightGBM 显著优于传统基线模型**：
   - MAE 降低 32% (vs Naive)
   - MAPE 降低 48% (vs Naive)
   - 方向预测准确率提升 12 个百分点

3. **当前波动率是最重要的预测特征**，表明波动率具有持续性

4. **滚动窗口波动率的目标本质平滑**，导致 R² 不是可靠的评估指标

5. **跨指数泛化能力有限**：
   - 对大盘蓝筹指数（上证50、沪深300）泛化良好
   - 对创业板泛化能力差，需单独建模

### 7.2 实践建议

1. 使用 **MAE/MAPE** 作为波动率预测的主要评估指标
2. 关注 **方向准确率**，对交易策略有直接指导意义
3. 优先使用 **Yang-Zhang 波动率** 作为波动率估计量
4. 避免过度依赖 EWMA，其对波动变化响应过慢
5. **仅对相似特性的指数直接应用模型**，创业板等需单独训练

### 7.3 后续改进方向

1. 为不同风格指数训练专用模型
2. 加入宏观经济特征（VIX、利率、汇率等）
3. 探索深度学习模型（LSTM、Transformer）
4. 开发波动率交易策略并回测
